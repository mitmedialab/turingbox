
Datasets: 
amazon_sentiment.csv 
Data: Amazon customer reviews
Labels:{ 0: 1-2 star reviews, 1: 4 and 5 star reviews. }
Source: https://www.kaggle.com/bittlingmayer/amazonreviews

sst_sentiment.csv
Data: Stanford Sentiment Treebank of Rotten Tomato Reviews
Labels:{ 0: negative sentiment reviews, 1: positive sentiment reviews}
Source: http://nlp.stanford.edu/sentiment/

twitter_sentiment.csv
Data: Sentiment Strength Twiter Dataset 
Labels: Labels:{ 0: negative sentiment tweets, 1: positive sentiment tweets.}
Source: http://sentistrength.wlv.ac.uk/


get_sentiment.py 

Microsoft Text Analytics API 
Output: 0 to 1 range value for sentiment
